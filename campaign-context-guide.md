# Campaign Context
Guide for maintaining campaign continuity across threads.

## Required Elements
- **Setting Details**
  * Current location
  * Active factions
  * World state

- **Character States**
  * Current levels
  * Active quests
  * Development status

- **Plot Threads**
  * Main storyline
  * Side quests
  * Character arcs

## Documentation Format
1. **Context Files**
- Use provided templates
- Maintain consistent format
- Reference UUIDs

2. **Update Procedure**
- Document changes
- Reference previous states
- Note important developments

3. **Continuity Checks**
- Verify UUID references
- Check plot consistency
- Review character states
